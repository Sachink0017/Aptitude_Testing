The name of the projectc is "AI-BASED APTITUDE TESTING SYSTEM FOR EARLY EDUCATION".
It is the system that is adaptive which means whenever any user runs the test, it adapts the difficulty of the test in real time, the difficulty levels are Easy, Medium, and Hard.
The adjustment of the difficulty is done with the help of a model which is made with Python and its libraries like Gymnasium, pandas.
The frontend of the project is developed with react,js and chakraUI framework of react.
